# OpenSSH ## OSX Enabling `ecdsa` encryption on Yosemite, install `openssh`. ### Install prerequisity : `xcode` and `homebrew` ~~~ $ brew update $ brew tap homebrew/dupes $ brew isntall openssh --with-brewed-openssl --with-keychain-support ~~~ ### Configure configurations on `openssh`es ~~~ # vim /System/Library/LaunchAgents/org.openbsd.ssh-agent.plist /usr/bin/ssh-agent #BEFORE /usr/local/bin/ssh-agent #AFTER ~~~ ~~~ $ launchctl unload -w /System/Library/LaunchAgents/org.openbsd.ssh-agent.plist $ launchctl load -w -S Aqua /System/Library/LaunchAgents/org.openbsd.ssh-agent.plist ~~~ `~/.bash_profile` ~~~ eval $(ssh-agent) function cleanup { echo "Killing SSH-Agent" kill -9 $SSH_AGENT_PID } ~~~
