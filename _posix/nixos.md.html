--- title: NixOS layout: dep --- # Installation 1. Download "MINIMAL INSTALLATION CD 64 bit" NixOS image, Write it into USB, and boot it. * From https://nixos.org/nixos/download.html
2. Connect Network via WIFI ~~~sh INTERFACE=wlp4s0 wpa_supplicant -B -i $INTERFACE -c <(wpa_passphrase 'SSID' 'key') systemctl restart wpa_supplicant ~~~ 3. Partitioning : Let's say you do; - make GRUB install and enable HIBERNATE - encrypt your storage ~~~sh lsblk ## find your device DISK="/dev/xxx" ## set xxx as your device ~~~ * Use `fdisk`; - At NixOS, GRUB can be installed on MBR. - To build MBR partitioning, use `fdisk` - if you use `system-boot` on GPT then use `gdisk` ~~~sh fdisk $DISK ## fdisk do partitioning over MBR ## o => ## '=>' denotes 'Enter key' ## n => p => => +512M => ## create MBR dir ## n => p => => => => ## ## w => ~~~ 4. encrypt the volume `${DISK}2` and open the volume with naming it `enc-pv` ~~~sh cryptsetup luksFormat ${DISK}2 #type YES then set PASSWORD cryptsetup luksOpen ${DISK}2 enc-pv ~~~ 5. LVM: The following command will create `root` and `swap` lvm on `vg` ~~~sh pvcreate /dev/mapper/enc-pv vgcreate vg /dev/mapper/enc-pv lvcreate -n swap vg -L 16G lvcreate -n root vg -l 100%FREE ~~~ * ( In case you reboot, you may want to re-mount these volumes with; ) ~~~sh ## vgchange -a y vg ## In case This activate vg in case you reboot ~~~ 6. Make File system over these partitions; ~~~sh mkfs.ext4 -L boot ${DISK}1 mkfs.ext4 -L root /dev/vg/root mkswap -L swap /dev/vg/swap ~~~ 7. Mount; ~~~sh mount /dev/vg/root /mnt mkdir /mnt/boot mount ${DISK}1 /mnt/boot ~~~ 8. Turn on Swap ~~~sh swapon /dev/vg/swap ~~~ 9. Generate an initial configuration; ~~~sh nixos-generate-config --root /mnt ~~~ 10. Edit Config File `/mnt/etc/nixos/configuration.nix`, or use my particular configuration with; ~~~sh ## curl http://raw.githubusercontent.com/ghassheee/nixos/master/configuration.nix > /mnt/etc/nixos/configuration.nix ~~~ 11. Install NixOS; ~~~sh nixos-install ~~~ 12. reboot ~~~sh reboot ~~~ 13. Do not forget to set PASSWORD for users ~~~sh passwd ghasshee ## i.e. set password of your username for login ~~~ That's all!
